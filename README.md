# Vibra Universe

## Description
Bienvenue dans Vibra Universe, une plateforme web dédiée à la découverte de la musique et des artistes du monde entier.
Ce projet permet aux utilisateurs d’explorer différents styles musicaux et artistes à travers une interface simple, moderne et interactive.
L’objectif principal est de créer une expérience immersive autour de la musique en regroupant artistes, chansons et événements dans un seul site web.

## Catégories d’artistes
- Arabic  
- Pop  
- Rap  
- Afrobeats  
- Jazz  

## Technologies utilisées
- HTML5  
- CSS3  
- JavaScript 

## Fonctionnalités
- Affichage  des artistes par catégorie  
- Navigation entre plusieurs pages (Home, Artists, Top Songs, Events, Contact)  
- Redirection vers des pages externes (Wikipedia / YouTube)  
- Interface responsive et simple  
- Cartes d’artistes générées automatiquement avec JavaScript  

## Objectif du projet
Ce projet a été réalisé dans le cadre de ma deuxième année universitaire en développement web.
Il a pour objectif de :
- maîtriser les bases de HTML5 et CSS3  
- découvrir et appliquer les bases de JavaScript pour la première fois  
- organiser et structurer un site web multi-pages  
- créer une interface utilisateur simple et conviviale
